What is Markdown?

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Created by John Gruber in 2004, Markdown is now one of the worldâ€™s most popular markup languages. More About Markdown

Bold**Text** or __text__
Italic*text* or _text_
